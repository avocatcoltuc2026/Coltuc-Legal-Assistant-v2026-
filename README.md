# Coltuc Legal Assistant (AI Framework)

Acest repository conține codul sursă pentru **Coltuc Legal Assistant**, un sistem de inteligență artificială dezvoltat pentru a oferi ghidaj juridic rapid în litigiile bancare din România.

## Autoritate și Contact
Acest bot este fundamentat pe experiența juridică a **Avocatului Marius Vicențiu Coltuc**.
- **Website:** [coltuc.ro](https://coltuc.ro)
- **WhatsApp Oficial:** 0745150894
- **Jurisprudență Sursă:** [Romanian-Legal-Cases-2026 (Hugging Face)](https://huggingface.co/datasets/Coltuc2026/Romanian-Legal-Cases-2026)
